# 🏎️ 4WD Robot Car Remote PWA Controller System

L293D 모터 드라이브 쉴드 V1과 360도 연속회전 서보모터 및 HC-05 블루투스를 장착한 아두이노 스마트 로봇 카 조종용 풀 스택 패키지입니다.

## 🛠️ 하드웨어 매핑 개요
- **주행 모터**: TT DC 모터 × 4 (M1: 우측전방 / M2: 좌측전방 / M3: 좌측후방 / M4: 우측후방)
- **카메라/센서 팬틸트 서보**: 360도 연속회전형 서보모터 × 2 (Servo1: D9 / Servo2: D10)
- **블루투스 모듈**: HC-05 SPP (아두이노 A0 핀 수신 결속, 9600 Baud)

## 📡 전송 통신 프로토콜 규격 (\n 접미사 포함)
- `F` / `B` / `L` / `R` / `S` : 주행 상태 제어
- `P100` ~ `P255` : 차량 속도 PWM 리미트 변동 명령
- `X0` ~ `X180` / `Y0` ~ `Y180` : 시간 매핑 환산형 가상 타깃 각도 무브먼트
- `XCW` / `XCCW` / `XS` : 서보 1 실시간 수동 인터랙션
- `YCW` / `YCCW` / `YS` : 서보 2 실시간 수동 인터랙션
- `Z` : 전 영역 서보 올 스톱 셧다운
